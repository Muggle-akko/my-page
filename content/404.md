# 404 找不到页面

你似乎迷路了。请返回 [首页](/my-page/)。

<script>
  // 客户端重定向：当 404 页面加载时，尝试重定向到正确的路径
  // 这是解决 GitHub Pages 刷新 404 问题的经典 SPA 路由技巧。
  // 它将 /repo-name/path/to/note 重定向到 /repo-name/#/path/to/note
  (function(){
    var redirect = sessionStorage.getItem('redirect');
    if (redirect) {
      sessionStorage.removeItem('redirect');
      location.replace(redirect);
    } else {
      // 如果没有重定向信息，则尝试将当前路径作为哈希路由访问
      var path = location.pathname.substring(1); // 移除开头的斜杠
      location.replace(location.origin + location.pathname + '#/' + path);
    }
  })();
</script>
